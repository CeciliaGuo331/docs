## VitePress 特色容器

VitePress 提供了一些预设样式的容器，可以突出显示特定信息。

::: info
这是一个信息容器。适合放置中性的提示信息。
:::

::: tip
这是一个提示容器。适合放置建议性的内容。
:::

::: warning
这是一个警告容器。适合放置需要注意的重要信息。
:::

::: danger
这是一个危险容器。适合放置非常重要或危险的警告信息。
:::
